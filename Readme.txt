Application: Measurements visualization 
Developed on Ubuntu 20.04 using Qt5/QtCreator/xlib/C++. 

Functionality description:
1 To execute, enter command './chart'
2 Open menu and select a *.ssd/*.rsd file to visualize.
3 Program can detect an error while parsing a file, in this case it shows a number of a line with the error. An empty line is not considered as an error.
4 To exit, open menu and click 'exit'.
